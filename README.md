# 🌍 React Travel Booking App

A modern and responsive **React Travel Booking Application** that allows users to search destinations, explore hotels, and view popular travel places through a clean and user-friendly interface.

This project demonstrates real-world React concepts such as **component-based architecture, dynamic routing, reusable components, and conditional rendering**, while maintaining a professional UI built with Tailwind CSS.

## 🛠️ Features

- 🔍 Search destinations  
- 🧭 Dynamic routing using useParams  
- 🏨 View hotels with ratings and pricing  
- 📍 Explore popular places in each destination  
- ❌ Displays "No Results Found" for unsupported searches  
- ♻️ Reusable components architecture  
- 📱 Fully responsive design  
- 🎨 Clean modern UI built with Tailwind CSS  

---

## ⚙️ Tech Stack

- React JS  
- Vite  
- Tailwind CSS  
- React Router DOM  
- JavaScript (ES6+)  
- Component-Based Architecture
